# app_calorias_diarias

## Aplicativo para calculo de calorias a serem consumidas diariamente de acordo com o nível de atividade física do individuo. 

O aplicativo tem o intuito de facilitar o calculo calorico das refeições e trazer uma maior noção a respeito de educação alimentar e estilo de vida
saudável já que através das informações individuais calcula a quantidade necessária de calorias diárias e a divisão equilibrada dos macronutrientes (carboidrato, proteina e gordura).
